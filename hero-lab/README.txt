# Hero Counter + Hero Synergy Prototype

Standalone prototype for testing the design and user experience before connecting it to SaifKS.com.

Files:
- index.html
- styles.css
- app.js

How to test:
1. Extract the ZIP file.
2. Open index.html in Safari, Chrome, or Edge.
3. Select a hero and battle mode.
4. Press Analyze Matchup.

Notes:
- This version is not connected to the main website.
- Hero data is sample data for testing the interface and behavior.
- Real hero images and final verified game data can be added after design approval.
